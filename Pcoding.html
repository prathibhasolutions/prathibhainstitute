Python Programming Tasks

Basics

1. Write a Python program to print "Hello, World!"
2. Write a program to take user input and print it.
3. Write a program to swap two variables without using a temporary variable.
4. Write a program to check if a number is positive, negative, or zero.
5. Write a program to calculate the area of a circle given its radius.

Operators

6. Write a program to demonstrate the use of arithmetic operators (+, -, *, /, %, //, **).
7. Write a program to compare two numbers using relational operators and print the results.
8. Write a program to use logical operators (and, or, not) with boolean variables.
9. Write a program to demonstrate assignment operators (+=, -=, *=, /=).
10. Write a program to check if a number is even or odd using the modulus operator.

Data Types

11. Write a program to find the type of variables holding int, float, string, and boolean values.
12. Write a program to convert a string to integer and float and demonstrate errors when conversion fails.
13. Write a program to concatenate two strings and print the result.
14. Write a program to demonstrate typecasting between int, float, and string.
15. Write a program to create a list and print its length and elements.

If-Else Statements

16. Write a program to find the largest of three numbers using if-else statements.
17. Write a program to check if a character is a vowel or consonant.
18. Write a program to determine if a year is a leap year or not.
19. Write a program to calculate the grade of a student based on marks using if-elif-else.
20. Write a program to check whether a number is divisible by 3 and 5 both.

Loops

21. Write a program to print numbers from 1 to 10 using a for loop.
22. Write a program to calculate the factorial of a number using a while loop.
23. Write a program to print the multiplication table of a number using a for loop.
24. Write a program to print all the even numbers between 1 and 50 using a loop.
25. Write a program to find the sum of all numbers in a list using a loop.

Pattern Printing

26. Write a program to print the following pattern:

*
* *
* * *
* * * *
* * * * *

27. Write a program to print a right-aligned triangle pattern of stars:

        *
      * *
    * * *
  * * * *
* * * * *

28. Write a program to print a number pyramid:

1
2 2
3 3 3
4 4 4 4

29. Write a program to print an inverted triangle pattern:

* * * * *
* * * *
* * *
* *
*

30. Write a program to print a hollow square of stars (n = 5):

* * * * *
*       *
*       *
*       *
* * * * *
