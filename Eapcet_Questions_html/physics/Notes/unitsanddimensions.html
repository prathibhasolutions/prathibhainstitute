Units and dimensions

1. Physical Quantities

Physical quantities are measurable properties of a system that can be expressed in terms of a unit.

They are classified into:

  • Fundamental Quantities: Cannot be derived from other quantities.
  • Derived Quantities: Formed by combining fundamental quantities.
2. Units of Measurement

A unit is a standard quantity used to measure physical quantities.

Types of Units:

  • Fundamental Units: Defined for fundamental quantities.
  • Derived Units: Expressed using fundamental units.
  • SI Units: Internationally accepted system of units.

Principle of Homogeneity of Dimensions

The principle of homogeneity states that in a physically correct equation, the dimensions of all terms on both sides must be the same.

Mathematical Representation

If an equation is of the form:

\( A + B = C \)

Then, the dimensions of A, B, and C must be the same:

\( [A] = [B] = [C] \)

Applications of Homogeneity Principle
  • Checking correctness of an equation.
  • Deriving unknown relations between physical quantities.
  • Conversion of units between different systems.
  • Ensuring dimensional consistency in derived equations.
Examples
  • Example 1: Kinematic equation

    • \( s = ut + \frac{1}{2} a t^2 \)

    Checking dimensions:

    \( [s] = [ut] = [\frac{1}{2} a t^2] = L \) ✅

  • Example 2: Force equation

    • \( F = ma \)

    \( [F] = [m] [a] = M (LT^{-2}) = MLT^{-2} \) ✅

  • Example 3: Energy equation

    • \( E = \frac{1}{2} m v^2 \)

    \( [E] = [m] [v^2] = M (L^2 T^{-2}) = ML^2T^{-2} \) ✅

Significant Figures

Significant figures are the digits in a number that contribute to its accuracy. They include all known digits plus the first uncertain digit.

Rules for Counting Significant Figures
  • Non-zero digits are always significant. (e.g., 432 has 3 significant figures)
  • Zeros between non-zero digits are significant. (e.g., 405.6 has 4 significant figures)
  • Leading zeros (before the first non-zero digit) are NOT significant. (e.g., 0.0032 has 2 significant figures)
  • Trailing zeros in a decimal number are significant. (e.g., 4.500 has 4 significant figures)
  • Trailing zeros in a whole number without a decimal point are NOT significant. (e.g., 4500 has 2 significant figures, but 4500.0 has 5)
  • Scientific notation: Only the digits in the coefficient are significant. (e.g., \( 3.45 \times 10^5 \) has 3 significant figures)
Significant Figures in Algebraic Operations
  • Addition & Subtraction: The result should have as many decimal places as the number with the least decimal places.

    • \( 12.345 + 4.6 = 16.9 \) (since 4.6 has only 1 decimal place)

  • Multiplication & Division: The result should have as many significant figures as the number with the least significant figures.

    • \( 4.56 \times 2.1 = 9.6 \) (since 2.1 has only 2 significant figures)

  • Rounding Off:
    • If the digit to be dropped is less than 5, keep the previous digit the same.
    • If the digit to be dropped is greater than 5, increase the previous digit by 1.
    • If the digit to be dropped is exactly 5, round to the nearest even number.

    • Example: \( 3.745 \) rounded to 3 significant figures → \( 3.74 \)

Examples
  • How many significant figures?

    • 0.00560 → 3 significant figures

    12000 → 2 significant figures

  • Applying Rules:

    • \( 5.678 + 0.42 = 6.10 \) (rounded to 2 decimal places)

    \( 7.25 \times 0.004 = 0.029 \) (rounded to 2 significant figures)

Rounding Off and Rules

Rounding off is the process of reducing the number of digits while keeping the value close to the original.

Rules for Rounding Off
  • If the digit to be removed is < 5, the preceding digit remains unchanged.

    • Example: 3.642 → 3.64 (rounded to 2 decimal places)

  • If the digit to be removed is > 5, increase the preceding digit by 1.

    • Example: 4.678 → 4.68 (rounded to 2 decimal places)

  • If the digit to be removed is exactly 5:
    • If the preceding digit is even: It remains unchanged.

      • Example: 3.450 → 3.4 (rounded to 1 decimal place)

    • If the preceding digit is odd: It is increased by 1.

      • Example: 2.750 → 2.8

Examples
  • Rounding to 2 decimal places:

    • 5.786 → 5.79

    4.672 → 4.67

  • Rounding to 1 decimal place:

    • 9.45 → 9.4 (preceding digit even)

    7.75 → 7.8 (preceding digit odd)

Errors and Their Classification
Definition

Error is the difference between the measured value and the true value of a quantity.

Types of Errors
  • Systematic Errors: Occur due to faulty instruments, calibration errors, or environmental factors.
  • Random Errors: Occur due to unpredictable variations in measurement.
  • Gross Errors: Caused by human mistakes like incorrect readings.
Absolute Error

Absolute error is the difference between each measured value and the true value.

\[ \Delta X_i = |X_i - X_{\text{true}}| \]

Example: If true value = 50 cm, measured values = 49 cm, 51 cm, 50.5 cm, then absolute errors are 1 cm, 1 cm, and 0.5 cm.

Mean Absolute Error

Mean absolute error is the average of all absolute errors.

\[ \Delta X_{\text{mean}} = \frac{\sum |\Delta X_i|}{n} \]

Example: If absolute errors are 1 cm, 1 cm, and 0.5 cm, then:

\[ \Delta X_{\text{mean}} = \frac{1+1+0.5}{3} = 0.83 \text{ cm} \]

Relative Error

Relative error is the ratio of the mean absolute error to the true value.

\[ \text{Relative Error} = \frac{\Delta X_{\text{mean}}}{X_{\text{true}}} \]

Example: If \(\Delta X_{\text{mean}} = 0.83\) cm and \(X_{\text{true}} = 50\) cm, then:

\[ \text{Relative Error} = \frac{0.83}{50} = 0.0166 \]

Percentage Error

Percentage error is the relative error expressed in percentage.

\[ \text{Percentage Error} = \text{Relative Error} \times 100\% \]

Example: If relative error = 0.0166, then:

\[ \text{Percentage Error} = 0.0166 \times 100 = 1.66\% \]

Combination of Errors
Error in Addition or Subtraction

If \( Z = A \pm B \), then the absolute error in \( Z \) is given by:

\[ \Delta Z = \Delta A + \Delta B \]

Example: If \( A = 20 \pm 0.5 \) and \( B = 30 \pm 0.3 \), then:

\[ \Delta Z = 0.5 + 0.3 = 0.8 \]

Error in Multiplication or Division

If \( Z = A \times B \) or \( Z = \frac{A}{B} \), then the relative error in \( Z \) is given by:

\[ \frac{\Delta Z}{Z} = \frac{\Delta A}{A} + \frac{\Delta B}{B} \]

Example: If \( A = 20 \pm 0.5 \) and \( B = 30 \pm 0.3 \), then:

\[ \frac{\Delta Z}{Z} = \frac{0.5}{20} + \frac{0.3}{30} = 0.025 + 0.01 = 0.035 \]

So, the percentage error = 3.5%.

Error when a Quantity is Raised to Power \( n \)

If \( Z = A^n \), then the relative error in \( Z \) is:

\[ \frac{\Delta Z}{Z} = n \times \frac{\Delta A}{A} \]

Example: If \( A = 5 \pm 0.1 \) and \( Z = A^3 \), then:

\[ \frac{\Delta Z}{Z} = 3 \times \frac{0.1}{5} = 0.06 \]

So, the percentage error = 6%.
