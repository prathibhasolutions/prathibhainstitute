Trigonometric Equations

Definition

A trigonometric equation is an equation that involves trigonometric functions like \( \sin x \), \( \cos x \), \( \tan x \), etc., and requires solving for values of \( x \).

Types of Solutions
1. General Solution
The general solution of a trigonometric equation gives all possible values of the variable that satisfy the equation.
Example: If \( \sin x = 0 \), the general solution is: \[ x = n\pi, \quad n \in \mathbb{Z} \]
2. Principal Solution
The principal solution consists of the solutions within a single cycle of the trigonometric function, typically in the interval \( [0, 2\pi] \) or \( (-\pi, \pi] \).
Example: For \( \sin x = \frac{1}{2} \), the principal solutions are: \[ x = \frac{\pi}{6}, \quad x = \frac{5\pi}{6} \]
3. Particular Solution
A particular solution is a specific solution of the equation within a given interval.
Example: Solve \( \tan x = 1 \) for \( x \in [0, \pi] \). The solution is \( x = \frac{\pi}{4} \).

General Solutions of Trigonometric Equations

Trigonometric Equation General Solution Domain of \( \alpha \)
\( \sin x = k \) \( x = n\pi + (-1)^n \alpha \) \( \alpha = \sin^{-1} k, \quad -\frac{\pi}{2} \leq \alpha \leq \frac{\pi}{2} \)
\( \cos x = k \) \( x = 2n\pi \pm \alpha \) \( \alpha = \cos^{-1} k, \quad 0 \leq \alpha \leq \pi \)
\( \tan x = k \) \( x = n\pi + \alpha \) \( \alpha = \tan^{-1} k, \quad -\frac{\pi}{2} < \alpha < \frac{\pi}{2} \)
\( \cot x = k \) \( x = n\pi + \alpha \) \( \alpha = \cot^{-1} k, \quad 0 < \alpha < \pi \)
\( \sec x = k \) \( x = 2n\pi \pm \alpha \) \( \alpha = \sec^{-1} k, \quad 0 \leq \alpha \leq \pi, \quad \alpha \neq \frac{\pi}{2} \)
\( \csc x = k \) \( x = n\pi + (-1)^n \alpha \) \( \alpha = \csc^{-1} k, \quad -\frac{\pi}{2} \leq \alpha \leq \frac{\pi}{2}, \quad \alpha \neq 0 \)

Solving Trigonometric Equations

Example 1: Solve \( \sin x = \frac{1}{2} \)

General Solution:

\[ x = n\pi + (-1)^n \alpha, \quad \alpha = \sin^{-1} k \]

Since \( \sin 30^\circ = \sin \frac{\pi}{6} = \frac{1}{2} \), we get:

\[ x = n\pi + (-1)^n \frac{\pi}{6}, \quad n \in \mathbb{Z} \]

Example 2: Solve \( \cos x = \frac{1}{2} \)

General Solution:

\[ x = 2n\pi \pm \alpha, \quad \alpha = \cos^{-1} k \]

Since \( \cos 60^\circ = \cos \frac{\pi}{3} = \frac{1}{2} \), we get:

\[ x = 2n\pi \pm \frac{\pi}{3}, \quad n \in \mathbb{Z} \]

Example 3: Solve \( \tan x = 1 \)

General Solution:

\[ x = n\pi + \tan^{-1} k \]

Since \( \tan 45^\circ = \tan \frac{\pi}{4} = 1 \), we get:

\[ x = n\pi + \frac{\pi}{4}, \quad n \in \mathbb{Z} \]

Example 4: Solve \( \cot x = \sqrt{3} \)

General Solution:

\[ x = n\pi + \cot^{-1} k \]

Since \( \cot 30^\circ = \cot \frac{\pi}{6} = \sqrt{3} \), we get:

\[ x = n\pi + \frac{\pi}{6}, \quad n \in \mathbb{Z} \]

Example 5: Solve \( \sec x = 2 \)

General Solution:

\[ x = 2n\pi \pm \sec^{-1} k \]

Since \( \sec 60^\circ = \sec \frac{\pi}{3} = 2 \), we get:

\[ x = 2n\pi \pm \frac{\pi}{3}, \quad n \in \mathbb{Z} \]

Example 6: Solve \( \csc x = -2 \)

General Solution:

\[ x = n\pi + (-1)^n \csc^{-1} k \]

Since \( \csc(-30^\circ) = \csc(-\frac{\pi}{6}) = -2 \), we get:

\[ x = n\pi + (-1)^n \left( -\frac{\pi}{6} \right) \]

\[ x = n\pi - (-1)^n \frac{\pi}{6}, \quad n \in \mathbb{Z} \]
