Properties of Triangle

Basic Properties of a Triangle
Sides of a Triangle

\( a, b, c \) represent the three sides of the triangle.

Angles of a Triangle

\( A, B, C \) represent the three interior angles of the triangle.

Sum of Angles in a Triangle

The sum of all three interior angles of a triangle is always \( 180^\circ \).

\( A + B + C = 180^\circ \)

Semi-Perimeter of a Triangle

The semi-perimeter \( s \) of a triangle is given by:

\( s = \frac{a + b + c}{2} \)

Area of a Triangle

The area of a triangle is given by:

\( \text{Area} = \frac{1}{2} \times \text{Base} \times \text{Height} \)

Sine Rule in a Triangle

The Sine Rule can be written using the circumradius \( R \):

\[ \frac{a}{\sin A} = \frac{b}{\sin B} = \frac{c}{\sin C} = 2R \]

Cosine Rule in a Triangle
Cosine Rule Formula

The Cosine Rule states that in any triangle:

\[ a^2 = b^2 + c^2 - 2bc \cos A \] \[ b^2 = a^2 + c^2 - 2ac \cos B \] \[ c^2 = a^2 + b^2 - 2ab \cos C \]

Cosine Rule for Angle Calculation

The angles in a triangle can also be found using:

\[ \cos A = \frac{b^2 + c^2 - a^2}{2bc} \] \[ \cos B = \frac{a^2 + c^2 - b^2}{2ac} \] \[ \cos C = \frac{a^2 + b^2 - c^2}{2ab} \]

Projection Rule in a Triangle

The Projection Rule states that in any triangle:

\( a = b \cos C + c \cos B \)

\( b = a \cos C + c \cos A \)

\( c = a \cos B + b \cos A \)

Napier's Analogy
First Formula

\[ \frac{\tan \frac{A - B}{2}}{\tan \frac{A + B}{2}} = \frac{a - b}{a + b} \]

Second Formula

\[ \frac{\tan \frac{B - C}{2}}{\tan \frac{B + C}{2}} = \frac{b - c}{b + c} \]

Third Formula

\[ \frac{\tan \frac{C - A}{2}}{\tan \frac{C + A}{2}} = \frac{c - a}{c + a} \]

Trigonometric Ratios of Half Angles of a Triangle
\( \sin \frac{A}{2} \)

\[ \sin \frac{A}{2} = \sqrt{\frac{(s - b)(s - c)}{bc}} \]

\( \sin \frac{B}{2} \)

\[ \sin \frac{B}{2} = \sqrt{\frac{(s - a)(s - c)}{ac}} \]

\( \sin \frac{C}{2} \)

\[ \sin \frac{C}{2} = \sqrt{\frac{(s - a)(s - b)}{ab}} \]

\( \cos \frac{A}{2} \)

\[ \cos \frac{A}{2} = \sqrt{\frac{s(s - a)}{bc}} \]

\( \cos \frac{B}{2} \)

\[ \cos \frac{B}{2} = \sqrt{\frac{s(s - b)}{ac}} \]

\( \cos \frac{C}{2} \)

\[ \cos \frac{C}{2} = \sqrt{\frac{s(s - c)}{ab}} \]

\( \tan \frac{A}{2} \)

\[ \tan \frac{A}{2} = \sqrt{\frac{(s - b)(s - c)}{s(s - a)}} \]

\( \tan \frac{B}{2} \)

\[ \tan \frac{B}{2} = \sqrt{\frac{(s - a)(s - c)}{s(s - b)}} \]

\( \tan \frac{C}{2} \)

\[ \tan \frac{C}{2} = \sqrt{\frac{(s - a)(s - b)}{s(s - c)}} \]

Area of a Triangle - All Formulas
Basic Formula

\[ \Delta = \frac{1}{2} \times \text{Base} \times \text{Height} \]

Heron's Formula

\[ s = \frac{a + b + c}{2} \] \[ \Delta = \sqrt{s(s - a)(s - b)(s - c)} \]

Using Sine Rule

\[ \Delta = \frac{1}{2} ab \sin C \] \[ \Delta = \frac{1}{2} bc \sin A \] \[ \Delta = \frac{1}{2} ca \sin B \]

Using Circumradius (R)

\[ \Delta = \frac{abc}{4R} \]

Using Inradius (r)

\[ \Delta = sr \] (where \( s \) is the semi-perimeter)

Inradius (r) of a Triangle
Basic Formula

\[ r = \frac{\Delta}{s} \] (where \( s \) is the semi-perimeter)

Using Tangent Formula

\[ r = (s - a) \tan \frac{A}{2} \] \[ r = (s - b) \tan \frac{B}{2} \] \[ r = (s - c) \tan \frac{C}{2} \]

Using Half-Angle Formula

\[ r = 4R \sin \frac{A}{2} \sin \frac{B}{2} \sin \frac{C}{2} \]

Exradii (r₁, r₂, r₃) of a Triangle - All Formulas
Basic \(r_1\)

\[ r_1 = \frac{\Delta}{s - a} \]

Basic \(r_2\)

\[ r_2 = \frac{\Delta}{s - b} \]

Basic \(r_3\)

\[ r_3 = \frac{\Delta}{s - c} \]

Half-Angle \(r_1\)

\[ r_1 = s\,\tan\frac{A}{2} \]

Half-Angle \(r_2\)

\[ r_2 = s\,\tan\frac{B}{2} \]

Half-Angle \(r_3\)

\[ r_3 = s\,\tan\frac{C}{2} \]

Half-Angle \(r_1\)

\[ r_1 = 4R \sin\frac{A}{2} \cos\frac{B}{2} \cos\frac{C}{2} \]

Half-Angle \(r_2\)

\[ r_2 = 4R \sin\frac{B}{2} \cos\frac{C}{2} \cos\frac{A}{2} \]

Half-Angle \(r_3\)

\[ r_3 = 4R \sin\frac{C}{2} \cos\frac{A}{2} \cos\frac{B}{2} \]

Equilateral Triangle (Side = 2) - All Parameters
Side Length (a)

\[ a = b = c = 2 \]

Sum of Angles

\[ A + B + C = 180^\circ, \quad A = B = C = 60^\circ \]

Semi-Perimeter (s)

\[ s = \frac{3a}{2} = 3 \]

Area (\(\Delta\))

\[ \Delta = \frac{\sqrt{3}}{4} a^2 = \sqrt{3} \]

Inradius (r)

\[ r = \frac{\Delta}{s} = \frac{\sqrt{3}}{3} = \frac{1}{\sqrt{3}} \]

Circumradius (R)

\[ R = \frac{a}{\sqrt{3}} = \frac{2}{\sqrt{3}} \]

Exradius \(r_1\)

\[ r_1 = \frac{\Delta}{s - a} = \frac{\sqrt{3}}{3 - 2} = \sqrt{3} \]

Exradius \(r_2\)

\[ r_2 = \frac{\Delta}{s - b} = \sqrt{3} \]

Exradius \(r_3\)

\[ r_3 = \frac{\Delta}{s - c} = \sqrt{3} \]
