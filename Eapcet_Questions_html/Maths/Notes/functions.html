Functions

Definition of a Function

In mathematics, a function is a relation between a set of inputs and a set of possible outputs, where each input is related to exactly one output.

A function is commonly written as f(x), where x is the input and f(x) is the output. Functions help in modeling relationships between quantities and are fundamental in algebra and calculus.

Domain (A)

The set of all possible input values.

Example: A = {1, 2, 3, 4}

Function Representation

f: A → B

A function maps elements from set A (Domain) to set B (Codomain).

Codomain (B)

The set containing all possible outputs.

Example: B = {a, b, c, d, e}

Range

The actual set of outputs obtained from applying f to elements of A.

Example: If f(1) = a, f(2) = b, f(3) = c, then Range = {a, b, c}

Types of Functions

One-One (Injective) Function

Each element in the domain (A) maps to a unique element in the codomain (B).

Example: f(x) = x + 1

Many-to-One Function

Two or more elements in the domain map to the same element in the codomain.

Example: f(x) = x²

Onto (Surjective) Function

Every element in the codomain has at least one pre-image in the domain.

Example: f(x) = 2x (if codomain is even numbers).

Into Function

At least one element in the codomain has no pre-image in the domain.

Example: f(x) = x² where Codomain is all real numbers.

Bijection (One-One & Onto)

A function that is both One-One and Onto.

Example: f(x) = x (Identity function).

Types of Functions

Constant Function

Always returns the same value.

Example: \( f(x) = c \)

Linear Function

Forms a straight line.

Example: \( f(x) = mx + b \)

Identity Function

Returns the input as output.

Example: \( f(x) = x \)

Quadratic Function

Forms a parabola.

Example: \( f(x) = ax^2 + bx + c \)

Power Function

Has the form \( f(x) = x^n \).

Example: \( f(x) = x^3 \)

Square Root Function

Returns the square root of x.

Example: \( f(x) = \sqrt{x} \)

Modulus Function

Gives absolute value of x.

Example: \( f(x) = |x| \)

Signum Function

Returns -1, 0, or 1 based on sign of x.

Example: \( f(x) = \text{sign}(x) \)

Greatest Integer (Step) Function

Rounds down to the nearest integer.

Example: \( f(x) = \lfloor x \rfloor \)

Fractional Part Function

Extracts the decimal part of x.

Example: \( f(x) = x - \lfloor x \rfloor \)

Exponential Function

Has the form \( f(x) = a^x \).

Example: \( f(x) = 2^x \)

Logarithmic Function

Inverse of an exponential function.

Example: \( f(x) = \log x \)

Composition of Two Functions

The composition of two functions \( f \) and \( g \) is denoted as \( (f \circ g)(x) \), which means applying \( g \) first and then applying \( f \).

Example 1

Given:

\( f(x) = x^2 + 3 \)

\( g(x) = 2x + 1 \)

Find \( (f \circ g)(x) \):

\( (f \circ g)(x) = f(g(x)) = f(2x + 1) \)

\( = (2x + 1)^2 + 3 \)

\( = 4x^2 + 4x + 1 + 3 \)

\( = 4x^2 + 4x + 4 \)

Example 2

Given:

\( f(x) = \sqrt{x} \)

\( g(x) = x^2 - 4 \)

Find \( (f \circ g)(x) \):

\( (f \circ g)(x) = f(g(x)) = f(x^2 - 4) \)

\( = \sqrt{x^2 - 4} \)

Inverse of a Function

The inverse of a function \( f(x) \), denoted as \( f^{-1}(x) \), is a function that reverses \( f(x) \). That is, if \( f(a) = b \), then \( f^{-1}(b) = a \).

To find \( f^{-1}(x) \):

  1. Replace \( f(x) \) with \( y \).
  2. Swap \( x \) and \( y \).
  3. Solve for \( y \), which gives \( f^{-1}(x) \).
Example 1

Given function:

\( f(x) = 2x + 3 \)

Find \( f^{-1}(x) \):

1. Let \( y = 2x + 3 \)

2. Swap \( x \) and \( y \): \( x = 2y + 3 \)

3. Solve for \( y \):

\( y = \frac{x - 3}{2} \)

So, \( f^{-1}(x) = \frac{x - 3}{2} \).

Example 2

Given function:

\( f(x) = \frac{1}{x} \)

Find \( f^{-1}(x) \):

1. Let \( y = \frac{1}{x} \)

2. Swap \( x \) and \( y \): \( x = \frac{1}{y} \)

3. Solve for \( y \):

\( y = \frac{1}{x} \)

So, \( f^{-1}(x) = \frac{1}{x} \).

Periodic Function

A function \( f(x) \) is called a periodic function if there exists a positive constant \( T \) such that:

\( f(x + T) = f(x) \) for all \( x \) in the domain of \( f \).

The smallest such positive value of \( T \) is called the fundamental period of the function.

Example 1: Sine Function

The sine function is periodic with period \( 2\pi \).

Function: \( f(x) = \sin x \)

Verification:

\( f(x + 2\pi) = \sin(x + 2\pi) = \sin x = f(x) \)

Thus, the period of \( \sin x \) is \( 2\pi \).

Example 2: Cosine Function

The cosine function is also periodic with period \( 2\pi \).

Function: \( f(x) = \cos x \)

Verification:

\( f(x + 2\pi) = \cos(x + 2\pi) = \cos x = f(x) \)

Thus, the period of \( \cos x \) is \( 2\pi \).

Example 3: Tangent Function

The tangent function is periodic with period \( \pi \).

Function: \( f(x) = \tan x \)

Verification:

\( f(x + \pi) = \tan(x + \pi) = \tan x = f(x) \)

Thus, the period of \( \tan x \) is \( \pi \).

Example 4: General Periodic Function

Any function of the form \( f(x) = \sin(kx) \) or \( f(x) = \cos(kx) \) is periodic.

The period is given by:

\( T = \frac{2\pi}{k} \)

Example: \( f(x) = \sin(3x) \)

Period: \( T = \frac{2\pi}{3} \).

Even and Odd Functions

A function can be classified as even, odd, or neither based on its symmetry.

Even Function

A function \( f(x) \) is even if:

\( f(-x) = f(x) \) for all \( x \) in the domain.

Even functions are symmetric about the y-axis.

Example 1: \( f(x) = x^2 \)

\( f(-x) = (-x)^2 = x^2 = f(x) \), so it is even.

Example 2: \( f(x) = \cos x \)

\( f(-x) = \cos(-x) = \cos x = f(x) \), so it is even.

Odd Function

A function \( f(x) \) is odd if:

\( f(-x) = -f(x) \) for all \( x \) in the domain.

Odd functions are symmetric about the origin.

Example 1: \( f(x) = x^3 \)

\( f(-x) = (-x)^3 = -x^3 = -f(x) \), so it is odd.

Example 2: \( f(x) = \sin x \)

\( f(-x) = \sin(-x) = -\sin x = -f(x) \), so it is odd.

Neither Even Nor Odd

Some functions are neither even nor odd.

Example: \( f(x) = x^2 + x \)

\( f(-x) = (-x)^2 + (-x) = x^2 - x \)

\( f(-x) \neq f(x) \) and \( f(-x) \neq -f(x) \), so it is neither even nor odd.

Finding the Domain of a Function

The domain of a function \( f(x) \) is the set of all possible input values (values of \( x \)) for which the function is defined.

To find the domain, we check for:

  • Values that make the denominator zero (for rational functions).
  • Values that make the expression inside a square root negative (for square root functions).
  • Values that make the function undefined (such as logarithm of a non-positive number).
Example 1: Polynomial Function

Function: \( f(x) = x^2 + 3x + 5 \)

Polynomial functions are defined for all real numbers.

Domain: \( (-\infty, \infty) \)

Example 2: Rational Function

Function: \( f(x) = \frac{1}{x - 2} \)

The denominator cannot be zero, so we set:

\( x - 2 \neq 0 \Rightarrow x \neq 2 \)

Domain: \( (-\infty, 2) \cup (2, \infty) \)

Example 3: Square Root Function

Function: \( f(x) = \sqrt{x - 4} \)

The expression inside the square root must be non-negative:

\( x - 4 \geq 0 \Rightarrow x \geq 4 \)

Domain: \( [4, \infty) \)

Example 4: Logarithmic Function

Function: \( f(x) = \log(x + 1) \)

The argument of a logarithm must be positive:

\( x + 1 > 0 \Rightarrow x > -1 \)

Domain: \( (-1, \infty) \)

Finding the Range of a Function

The range of a function \( f(x) \) is the set of all possible output values (values of \( f(x) \)) corresponding to the domain.

To find the range, we analyze:

  • The function's behavior (increasing, decreasing, symmetry).
  • Any restrictions due to square roots, logarithms, or fractions.
  • Finding the inverse function \( x = f^{-1}(y) \) and solving for \( y \).
Example 1: Polynomial Function

Function: \( f(x) = x^2 \)

Since \( x^2 \) is always non-negative, the smallest value is \( 0 \) at \( x = 0 \).

Range: \( [0, \infty) \)

Example 2: Rational Function

Function: \( f(x) = \frac{1}{x} \)

As \( x \to \pm \infty \), \( f(x) \to 0 \), but \( f(x) \) never equals zero.

Range: \( (-\infty, 0) \cup (0, \infty) \)

Example 3: Square Root Function

Function: \( f(x) = \sqrt{x - 4} \)

The square root function is only defined for \( x \geq 4 \), and the smallest value is \( 0 \) when \( x = 4 \).

Range: \( [0, \infty) \)

Example 4: Logarithmic Function

Function: \( f(x) = \log(x) \)

The logarithm function is defined for \( x > 0 \) and takes all real values.

Range: \( (-\infty, \infty) \)
